SE Basic IV 4.2 Cordelia

An open source firmware for the Chloe 280SE microcomputer.

Online user guide

Key features in SE Basic IV 4.2 Cordelia

  • Rewritten from the ground up to target the Chloe 280SE microcomputer.
  • Will also run on a ZX-Uno, ZX-Dos, ZX-Prism, or ZX-Speectrum 128 with SLAM+ ULA and divIDE/divMMC interface.
  • On-entry syntax checking.
  • Expression evaluation constrained only by memory.
  • Powerful string slicing.
  • Microsoft BASIC syntax.
  • Localization support including code pages.
  • Direct support in BASIC for disk commands.
  • Full size (80 x 24 character) terminal.
  • Fully commented source code.
  • Directly enter characters 32 to 255 using Control and Meta shifts..
  • Modular assembly listing.
  • Refactored code for improved readability.
  • Support for up to 134 commands and functions.
  • All 256 characters are user definable.
  • Abbreviated command entry: GO. 100.
  • All symbols accessed directly with symbol shift.
  • ANSI BASIC-78 / ECMA-55 Minimal BASIC standard compliant.
  • Assembly compiles with Zeus.
  • Assembly file includes X80 virtual co-processor definitions.
  • Based on the Timex T/S 1000 ROM by John Grant & Steve Vickers.
  • Chloe font specially designed for maximum legibility in text mode.
  • Extended editng keys (Insert, Delete, Home, End, Pg Up, Pg Dn).
  • Cold restart clears all RAM.
  • Cursor jumps to an error in the line.
  • Cursors can move up and down in an edit line.
  • Enter numbers in binary, octal or hexadecimal format.
  • Default high-contrast white on blue display.
  • Default EGA 16-color palette.
  • Edit any line: EDIT 10
  • Enhanced floating point library.
  • Enter tab stops directly.
  • Keyboard buffer.
  • Fast reset.
  • Fully open source (GPLv3 licensed).
  • Includes legally machine reverse engineered replacements for critical routines.
  • Incorporates elements of SAM BASIC by Andy Wright, creator of BETA BASIC.
  • Increased line range: 0-16383.
  • IY register available in interrupt mode 1
  • More room for BASIC programs.
  • Newton-Raphson square root function.
  • No keyword locations to memorize.
  • No redundant legacy code.
  • Optimized for speed.
  • Prevents accidental deletion of edit line on older keyboards.
  • Quick entry for common functions.
  • Recover from crashes with NMI.
  • Remove multiple lines: DELETE 10,100.
  • Set the color pair in text mode.
  • Scroll counter increased.
  • Shift in CAPS mode gives lower case letters.
  • Standard error messages.
  • Simplified flags.
  • Simplified key tables.
  • Single 23K BASIC program does not require ROM swapping..
  • Text after a REM statement is not tokenized.
  • Tokenized command entry.
  • Use REM at the start of a line to clear an unwanted edit line.
  • Extended system variables accessible using IY register.
  • Works with esxDOS on divIDE and divMMC.

SE Basic IV 4.2 Cordelia includes over 12,000 lines of code, and if developed in a conventional proprietary way would have required many years of work and cost over $500,000.

SE Basic IV 4.2 Cordelia is maintained by Cheveron Group.
